# Famories(ファモリーズ)

## サイト概要
### サイトテーマ
家族(**ファミリー**)や友達と、日々の記憶(**メモリー**)を**アルバムのように記録として残すことができるコミュニティサイトです。**

### テーマを選んだ理由
<課題>  
独立すると、家族や友達との思い出を作る機会や、どのように生活を送っているかを知ることが減ってしまいます。コロナ禍においては、より一層減ってしまったため、このようなことを解決したいと思いこのテーマを選びました。  
<解決方法>  
日々の記憶を共有できるサイトがあれば、大切な人達を身近に感じられることで課題を解決できると考えました。また、このサイトで投稿したものから出会いが生まれることで、家族のような新しいコミュニティを築くきっかけになると嬉しいと思いました。

### ターゲットユーザー
- 家族と離れて暮らしている人とその家族
- 友達と離れて暮らしている人とその友達
- 思い出を大切にしている人
- 趣味を大切にしている人
- 忘れごとが多い人
- 記録するのが好きな人
- 植物を栽培している人
- 動物を飼育している人
- イベントが好きな人
- スポーツが好きな人
- 旅行に行く人
- 誰かの投稿を見たい人など

### 主な利用シーン
- 日常の出来事を共有したいとき（楽しいこと、辛いこと、暇なことなどなんでも）
- 忘れたくない出来事があったとき(例、ペットのかわいいシーン）
- 趣味の記録を残したいとき（例、魚が釣れた）
- 忘れないように記録したいとき（例、夕飯の献立やレシピ、よく買う物を記録しておく）
- 趣味を共有したいとき（例、同じ趣味の共有、マイナースポーツを広める）
- 成長日記を残したいとき（例、子どもが初めて歩いた日、植物やペットの成長変化）
- 気持ちを共有したいとき（例、楽しいこと、辛いこと、暇なことなどなんでも）
- 誰かの投稿を見たいときなど

### 他サイトとの差別化
- TwitterやInstagramのグループはDM機能であり、本サイトはグループ投稿機能であること。
- LINEのLINE VOOMは動画配信へのフォロー機能であり、本サイトは画像を共有でき、そこから新しいコミュニティに発展できること。
- Facebookは全世界で利用されているが、本サイトは日本人向けにニックネームで利用できること。

#### 用語
- 『メモリー』　＝　『投稿するモノ』
- 『グループメモリー』　＝　『グループ内で投稿するモノ』
- ※グループ内ではメモリーと短縮している場合もあります
- 『メモ』　＝　『投稿したメモリー/グループメモリーの紹介やメモ書きとしての文章』

#### 注意点
- ゲストログインはメモリーの『閲覧』と『いいね』のみ可能
- グループ外で投稿したメモリーはメモリー一覧に公開される
- グループ内で投稿したグループメモリーはグループ内でのみ閲覧可能
- 本サイトにおいて公序良俗に反する利用はしないでください

#### 利用手順
- 閲覧といいね
  - 閲覧といいねは『ゲストログイン』でも可能です。
  - 『メモリー一覧』から投稿されたメモリーを閲覧することができます。
  - 『タグ一覧』から気になったタグを選択すると、該当するメモリーの一覧が閲覧できます。
  - 各メモリーを選択すると『メモリー詳細』が表示され、気に入ったメモリーに『いいね』ができます。
  - 『マイいいね』から『いいね』したメモリーの一覧が閲覧できます。
  - 『グループ一覧』からグループの一覧が閲覧できます。
  - 検索フォームから『メモリー』または『グループ』を選択し、キーワードに該当した一覧を表示できます。
- 新規登録とログイン
  - 『新規登録』するとすべての機能が利用できます。
  - 『新規登録』は、ニックネーム、メールアドレス、パスワードが必要です。
  - 『ログイン』は、メールアドレス、パスワードが必要です。
- メモリーの作成と投稿
  - 『メモリー作成』からメモリーの作成画面に遷移して作成することができます。
  - メモリーには画像、タイトル(必須)、メモ、タグを付けることができます。
  - タグは複数付けることもできます。
  - 『メモリー投稿』から投稿できます。
- コメント
  - 自身のメモリーや他の人のメモリーに『コメント』をすることができます。
  - 各メモリーの詳細画面にはコメント入力欄があり、コメント入力後に『コメント投稿』ができます。
- グループ
  - 『グループ作成』をしてみましょう。作成者がグループの『オーナー』となります。
  - 作成後は『グループページ』に遷移します。
  - マイページ等のユーザーインフォメーションからいつでもグループページに遷移できるようになります。
  - 『グループ一覧』から他のグループに加入リクエストを申請することもできます。
  - 『オーナー』はオーナーであるグループの『メンバー編集』が可能です。
  - 『メンバー編集』では、メンバーの退会処理と加入リクエストの『承認』または『拒否』ができます。
  - グループ内では、『グループメモリー作成』をすることができます。
  - グループ内のメモリーには「いいね」をすることができます。
  - 退会したい場合はグループページの『退会する』からできます。その後に表示される注意事項をよく読んでください。
  - ※現在オーナーは退会することができません。

## 実装機能
|機能名|実装理由|
|-|-|
|ゲストログイン機能|会員登録前にまずは試しに使えるようにし、登録のハードルを下げるため|
|会員機能|会員ごとに投稿、いいね、コメント、グループなどの情報を表示するため|
|投稿機能|会員が発信したい内容を投稿できるようにするため|
|投稿検索機能(キーワード検索)	|気になっているトピックに関する投稿をチェックできるようにするため|
|投稿検索機能(タグ検索)|気になっているカテゴリに関する投稿をチェックできるようにするため|
|いいね(ブックマーク)機能|気に入った投稿や留めておきたい投稿に対してリアクションするため|
|いいね(ブックマーク)一覧表示機能|いいね(ブックマーク)した投稿を好きな時に閲覧するため|
|コメント機能|気になった投稿に対してリアクションし、コミュニケーションを活発化させるため|
|グループ機能|グループ間での投稿、閲覧ができるようにするため|
|グループ加入機能(承認制)|プライベート空間であるグループに他の会員が自由に入れないようにするため|
|グループ投稿機能|グループ内での投稿をするため|
|グループ検索機能|気になっているグループや加入予定のグループを検索するため|
|グループ一覧表示機能|会員がグループ一覧を閲覧し加入リクエストを申請するため|
|管理機能（管理者）|会員、投稿、コメント、グループの一覧や登録内容を確認するため<br>不正な手段での登録や、不適切な会員、投稿、コメント、グループを削除する為|
